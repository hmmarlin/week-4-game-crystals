Welcome to Crystal Collector

You will be given a random number.

There are four crystals at the bottom of the page, each have a specific value assigned to them

Clicking on each of the crystals will give you points. If you match the random number exactly with the points, you win. If you go over, you lose.

Each time the game starts, the value of the crystals will change
